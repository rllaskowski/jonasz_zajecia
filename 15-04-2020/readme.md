## Tworzenie tablicy o określonej wielkości

```
# N - oczekiwana wielkość tablicy
# Tworzy tablicę o N elementach, wypełnioną zerami
tablica = [0]*N

# Przyklad: N = 5
tab = [0]*5

# Wypisze [0, 0, 0, 0, 0]
print(tab)
```
